# NotePro

A minimal, modern note-taking web app focused on speed, clarity, and everyday productivity.

## Overview

NotePro helps users capture ideas, organize notes, and quickly find what matters.
The interface is intentionally clean and distraction-free, with smooth interactions and practical note management features.

## Tech Stack

- React 18
- TypeScript
- Vite 5
- Tailwind CSS
- Radix UI primitives
- Framer Motion

## Getting Started

### Prerequisites

- Node.js 18+
- npm 9+

### Install

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

## Available Scripts

```bash
npm run dev      # Start development server
npm run build    # Create production build
npm run preview  # Preview production build locally
npm run lint     # Run ESLint
```

## Build for Production

```bash
npm run build
```

The output is generated in the dist folder.

## Project Structure
